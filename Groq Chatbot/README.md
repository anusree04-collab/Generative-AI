Groq AI Chatbot
---
Overview

This project demonstrates a simple AI chatbot built using the Groq API and a LLaMA-3.1 language model.
The chatbot takes user input from the terminal and generates an AI response in real time.

The purpose of this Proof of Concept (POC-2) is to understand how large language models can be integrated into applications using APIs.

---

Technologies Used

- Python
- Groq API
- LLaMA-3.1-8B Model
- Google Colab / Jupyter Notebook

---

How It Works

1. The user enters a question or message.
2. The program sends the input to the Groq API.
3. The LLaMA language model processes the request.
4. The AI response is returned and displayed to the user.

---

Installation

Install the Groq library:

pip install groq

---


Example Output

AI Chatbot Started (type 'exit' to stop)

Conclusion

This project demonstrates how conversational AI systems can be built using modern large language models.
It highlights the process of integrating AI APIs into Python applications for interactive communication.
